# Hello!

**CMD** (Cocoa Mark Down) is a lightweight Markdown *viewer* for macOS,
iPhone, and iPad.

## Key Features

- A simple file list and viewer layout
- Font size controls in **Settings** at the top right
- Light and dark appearance modes

## Supported Markdown

CMD supports basic syntax such as headings, lists, quotes, and code.

1. Ordered list
2. Second item
3. Third item

> Quotes are displayed like this.
> Multiple lines work too.

## Images

Both Markdown image syntax and `<img>` tags are supported.

Inline code like `let x = 42` and fenced code blocks are supported too.

```swift
struct Hello {
    let message = "Hello, CMD!"
}
```
