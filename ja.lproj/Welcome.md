# こんにちは！

**CMD** (Cocoa Mark Down) は、macOS、iPhone、iPad で使える
軽量なMarkdown *ビューア* です。

## 主な機能

- ファイル一覧とビューアだけのシンプルな構成
- 右上の **設定** から文字サイズを調整
- ライト / ダーク表示モードの切り替え

## 対応しているMarkdown

見出し、リスト、引用、コードなどの基本的な構文に対応しています。

1. 番号付きリスト
2. 2番目の項目
3. 3番目の項目

> 引用はこのように表示されます。
> 複数行にも対応しています。

## 画像

Markdownの画像構文と `<img>` タグの両方に対応しています。

![サンプル画像](https://picsum.photos/600/300)

<img src="https://picsum.photos/500/280" alt="HTML imgタグの例">

インラインコード `let x = 42` とコードブロックにも対応しています。

```swift
struct Hello {
    let message = "Hello, CMD!"
}
```
